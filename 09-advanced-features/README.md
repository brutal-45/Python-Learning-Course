# Module 09: Advanced Features

## 📋 Overview

This module covers advanced Python features including generators, iterators, context managers, and metaclasses.

## 🎯 Learning Objectives

- Create and use generators
- Implement custom iterators
- Build context managers
- Understand descriptors and metaclasses

|------|-------|-------------|
| [generators.py](generators.py) | Generators | yield keyword and generator expressions |
| [iterators.py](iterators.py) | Iterators | Custom iterator classes |
| [context_managers.py](context_managers.py) | Context Managers | with statement and custom managers |
